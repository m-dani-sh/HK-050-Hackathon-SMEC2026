# PriceMatch - E-commerce Price Comparison

A web application that compares prices across multiple e-commerce platforms to help users find the best deals.

## Features
- Search products across Amazon, Etsy, and Walmart
- Real-time price comparison from multiple sources
- Best deal highlighting and recommendations
- Product images and direct purchase links
- Responsive design with loading states

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- APIs: Etsy Open API, Amazon RapidAPI, Walmart SerpApi
- Design: Modern responsive layout
- Data: Real-time price fetching

## Setup
1. Add API keys to `script.js`:
   - FAKESTORE Key
   - RapidAPI Key (Amazon)
   - DUMMYSHOP API  
2. Open `index.html` in browser
3. Search for products to compare prices

## Usage
1. Enter product name in search box
2. View price comparisons across platforms
3. Identify best deals with highlighting
4. Click links to purchase from chosen platform

## Output
Displays product cards with prices, images, platform names, and best deal indicators for informed purchasing decisions.
